# 科创综指 MACD 择时信号

自动采集科创综指（000680.SH）行情数据，使用 MACD(12/18/13) 策略生成买卖信号，每日收盘后自动更新。

## 策略说明

- **指标**: MACD(12/18/13)，即快线12日EMA、慢线18日EMA、信号线13日EMA
- **买入**: DIF 从下方穿越 DEA（金叉）
- **卖出**: DIF 从上方穿越 DEA（死叉）
- **执行**: 信号日次日开盘价执行，扣除 0.1% 佣金 + 0.1% 滑点

## 策略表现（2020-01 至今）

| 指标 | MACD策略 | 买入持有 |
|------|---------|---------|
| 年化收益 | ~25% | ~13% |
| 最大回撤 | ~-27% | ~-58% |
| 夏普比率 | ~1.04 | — |

### 核心优势

- **熊市控损**: 2022年指数跌-25%，策略反而盈利+16%
- **盈亏比高**: 盈利交易平均+10.6%，亏损交易平均-3.0%，盈亏比3.48
- **参数鲁棒**: 通过过拟合检验（DSR p<10⁻⁷），非过拟合

## 自动更新

- GitHub Actions 每个交易日 15:30（北京时间）自动运行
- 采集最新日线和15分钟数据
- 生成信号并更新网页展示

## 信号展示页
